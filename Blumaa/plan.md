# Blumaa — Branding & Growth Agency Website

## Overview
A premium, multi-page website for Blumaa — a solo female-led branding and growth agency. Design is editorial, conversion-focused, and confident. No fluff, all strategy. Bilingual ES/EN. Always "I" / "yo" in copy — never "we".

---

## Design System v1.0

### Colors

| Name | Hex | Role |
|------|-----|------|
| Blumaa Blue | #2642FF | Primary hero — use freely as bg or text |
| Butter Yellow | #FFF3C4 | Primary hero — swaps with Blue |
| Pink | #FF62A1 | Accent only · logo star color |
| Citrus Orange | #FF8231 | Accent only |
| Pool Blue | #B6ECFF | Accent only |
| Crema | #FFF7EE | Light bg · logo text on dark — replaces pure white |
| Café | #452925 | Warm dark text |
| Night (Gris Oscuro) | #231F20 | Body text · darkest neutral |

**Core rules:**
- Blue ↔ Butter are the two hero colors — swap freely as bg/fg
- Max 1 accent color per section (Pink, Orange, Pool)
- Never use pure white #FFF — use Crema
- Never same color for card and its section background

**Approved pairings:**
- Blue bg + Crema text ✓
- Butter bg + Blue text ✓ (or Night text)
- Blue bg + Pink accent ✓
- Butter bg + Orange accent ✓
- Pink bg + Crema text ✓

**Card contrast rule:**
- Section Blue → cards in Butter or Crema
- Section Butter → cards in Blue or Crema (Crema card needs 1.5px Blue border)
- Section Crema → cards in Blue or Butter

### Typography

| Level | Font | Size | Weight | Case |
|-------|------|------|--------|------|
| H1 / Display | Instrument Serif | 48–72px | Regular | ALL CAPS |
| H2 / Heading | Instrument Serif | 28–40px | Regular | Flexible |
| H3 / Subheading | DM Sans | 18–20px | 500 | Flexible |
| Eyebrow / Label | Roboto Mono | 10–12px | 500 | UPPERCASE + spaced |
| Body | DM Sans | 14–17px | 400 | Sentence case |
| Handwritten | Biro Script | 18–24px | Regular | As-is, never CAPS |
| Button | DM Sans | 12–16px | 500 | Sentence case |

**Rules:**
- Max 2 font families per layout piece
- Biro Script: accent only, never for body text
- Roboto Mono: always UPPERCASE for labels and eyebrows
- H1 + key highlights → ALL CAPS

### Buttons — all pill shape (border-radius: 99px)

| Variant | Background | Text | Use |
|---------|-----------|------|-----|
| Primary | Butter #FFF3C4 | Blue #2642FF | Main CTA on dark sections |
| Dark | Blue #2642FF | Crema #FFF7EE | On light sections |
| Secondary | Crema #FFF7EE | Blue #2642FF | With 1.5px Blue border |
| Ghost | transparent | Crema #FFF7EE | With 1.5px Crema border, on Blue sections |

Hover: opacity 0.85 + translateY(-1px) · Transition: 200ms ease-in-out

### Form Inputs
- Input/Select: border-radius 99px · border 1.5px Blue · DM Sans 14px
- Textarea: border-radius 16px · same color rules
- Focus: box-shadow 0 0 0 3px rgba(38,66,255,0.12)
- Error: border-color Pink #FF62A1

### Cards (border-radius: 8px)
- Never same color as section background
- Crema card on Butter section → add 1.5px Blue border

### Spacing (8px base)
- Section padding: 64px (mobile) → 96px (desktop)
- Card padding: 24–32px
- Component gaps: 16–24px

### Border Radius
- Cards: 8px
- Buttons, inputs, tags: 99px (pill)
- Large containers: 16px

## Voice & Tone
- Bold · Direct · Witty · Irreverent · Human · Anti-corporate
- Always "I" / "yo" — never "we" / "nosotros"
- Name the pain first, then the solution
- Short sentences. The period is a weapon.
- Bilingual = real translation, same tone. Never literal.
- Banned words: passion, solutions, deliverables, holistic, synergy, comprehensive

---

## Key Design Principles
- Every section serves a purpose — no filler
- Blue ↔ Butter sections alternate for visual rhythm and contrast
- Cards always contrast with their section background
- CTAs are clear and conversion-focused throughout
- Mobile-responsive with the same editorial quality
